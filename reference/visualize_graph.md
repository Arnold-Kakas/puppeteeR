# Render a StateGraph visualization

Render a StateGraph visualization

## Usage

``` r
visualize_graph(graph, engine)
```

## Arguments

- graph:

  A
  [StateGraph](https://arnold-kakas.github.io/puppeteeR/reference/StateGraph.md)
  object.

- engine:

  Character. One of `"dot"`, `"visnetwork"`, `"mermaid"`.

## Value

A widget or prints to console.
